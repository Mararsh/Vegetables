661. Convert BST to Greater Tree

Given a Binary Search Tree (BST), convert it to a Greater Tree such that every key 
of the original BST is changed to the original key plus sum of all keys greater than
the original key in BST.

Example

Given a binary search Tree `{5,2,13}ï½€:

              5
            /   \
           2     13

Return the root of new tree

             18
            /   \
          20     13

Idea to solve the problem

Inorder traversal but right is first

My codes




/**
 * Definition of TreeNode:
 * public class TreeNode {
 *     public int val;
 *     public TreeNode left, right;
 *     public TreeNode(int val) {
 *         this.val = val;
 *         this.left = this.right = null;
 *     }
 * }
 */

public class Solution {
    /**
     * @param root: the root of binary tree
     * @return: the new root
     */
    // Total runtime 1883  ms. Your submission beats 87.40% Submissions!
     
    private int sum = 0;
    
    public TreeNode convertBST(TreeNode root) {
        if ( root == null ) return null;
        convertBST(root.right);
        root.val += sum;
        sum = root.val;
        convertBST(root.left);
        return root;
    }
}
