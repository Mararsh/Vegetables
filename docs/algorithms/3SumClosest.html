59. 3Sum Closest

Given an array S of n integers, find three integers in S such that the sum is closest to a given number, 
target. Return the sum of the three integers.
Example

For example, given array S = [-1 2 1 -4], and target = 1. The sum that is closest to the target is 2. 
(-1 + 2 + 1 = 2).

Challenge

O(n^2) time, O(1) extra space

Idea to solve the problem

Sorting/Two pointers

My codes




public class Solution {
    /**
     * @param numbers: Give an array numbers of n integer
     * @param target: An integer
     * @return: return the sum of the three integers, the sum closest target.
     */
    // Total runtime 2243 ms. Your submission beats 88.60% Submissions!
    public int threeSumClosest(int[] numbers, int target) {
        if ( numbers == null || numbers.length < 3 ) return -1;
        Arrays.sort(numbers);
        int diff = Integer.MAX_VALUE , dsum = -1;
        for (int i=0; i< numbers.length - 1; i++) {
            int left = i+1, right = numbers.length -1;
            while (left < right ) {
                int sum = numbers[i] + numbers[left] + numbers[right];
                if ( sum == target) {
                    return sum;
                } 
                int ndiff = Math.abs(sum -  target);
                if ( ndiff < diff ) {
                    diff = ndiff;
                    dsum = sum;
                }
                if ( sum > target ) {
                    right--;
                } else 
                    left++;
            }
        }
        return dsum;
    }
}
