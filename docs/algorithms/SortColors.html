148. Sort Colors

Given an array with n objects colored red, white or blue, sort them so that objects of the same color 
are adjacent, with the colors in the order red, white and blue.

Here, we will use the integers 0, 1, and 2 to represent the color red, white, and blue respectively.
Example

Given [1, 0, 1, 2], sort it in-place to [0, 1, 1, 2].
Challenge

A rather straight forward solution is a two-pass algorithm using counting sort.
First, iterate the array counting number of 0's, 1's, and 2's, then overwrite array with 
total number of 0's, then 1's and followed by 2's.

Could you come up with an one-pass algorithm using only constant space?

Idea to solve the problem

Move in-place

public class Solution {
    /**
     * @param nums: A list of integer which is 0, 1 or 2 
     * @return: nothing
     */
     // Total runtime 11810 ms. Your submission beats 85.80% Submissions!
     public void sortColors2(int[] nums) {
        if ( nums == null || nums.length == 1) return;
        int[] count = new int[3];
        for (int i=0; i< nums.length; i++ ) {
            count[ nums[i] ]++;
        }
        int p = 0;
        for (int i=0; i< 3; i++ ) {
            for (int j=0; j< count[i]; j++ ) {
                nums[p++] = i;
            }
        }
     }

    public void sortColors(int[] nums) {
        if ( nums == null || nums.length == 1) return;
        int p0=0, p2=nums.length-1, p1 = 0;
        while (p1 <= p2) {
            while ( p1 < p2 && nums[p2] == 2) p2--;
            if (nums[p1] == 2) {
                swap(nums, p1, p2);
                p2--;
            }
            while ( nums[p1] == 0 ) {
                swap(nums, p1, p0);
                p0++; 
                p1++;
            }
            while ( nums[p1] == 1 ) {
                p1++;
            } 
        }
        
    }
    
    
    public void swap(int[] d, int i, int j) {
        int tmp = d[i];
        d[i] = d[j];
        d[j] = tmp;
    }
