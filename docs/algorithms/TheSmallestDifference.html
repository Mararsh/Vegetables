387. The Smallest Difference

Given two array of integers(the first array is array A, the second array is array B), 
now we are going to find a element in array A which is A[i], and another element in array B which is B[j], 
so that the difference between A[i] and B[j] (|A[i] - B[j]|) is as small as possible, return their smallest 
difference.
Example

For example, given array A = [3,6,7,4], B = [2,8,9,3], return 0
Challenge

O(n log n) time

Idea to solve the problem

Sorting/Two pointers

My codes




public class Solution {
    /**
     * @param A: An integer array
     * @param B: An integer array
     * @return: Their smallest difference.
     */
    // Total runtime 4682 ms. Your submission beats 90.40% Submissions!
    public int smallestDifference(int[] A, int[] B) {
       Arrays.sort(A);
       Arrays.sort(B);
       int min = Integer.MAX_VALUE;
       int i =0, j =0;
       while ( i < A.length && j < B.length) {
           int diff = Math.abs(A[i] - B[j]);
           if ( diff == 0 ) return 0;       
           if (diff < min ) {
               min = diff;
           } 
           if ( A[i] > B[j] ) {
               j++;
           } else  {
               i++;
           }               
       }
       return min;
    }
}
