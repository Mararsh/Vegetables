363. Trapping Rain Water

Given n non-negative integers representing an elevation map where the width of each bar is 1, 
compute how much water it is able to trap after raining.

Trapping Rain Water
Example

Given [0,1,0,2,1,0,1,3,2,1,2,1], return 6.

Idea to solve the problem

Two pointers moving to center

public class Solution {
    /**
     * @param heights: a list of integers
     * @return: a integer
     */
    // Total runtime 1701 ms. Your submission beats 98.20% Submissions!
    public int trapRainWater(int[] heights) {
        if (heights == null || heights.length < 2 ) return 0;
        int lmax = heights[0], rmax = heights[heights.length - 1];
        int left = 1, right = heights.length -2;
        int count = 0;
        while ( left <= right ) {
            if (lmax > rmax ) {
                if ( heights[right] < rmax ) {
                    count += rmax -  heights[right];
                } else {
                    rmax = heights[right];
                }
                right--;
            } else {
                if ( heights[left] < lmax ) {
                    count += lmax-heights[left];
                } else {
                    lmax= heights[left];
                }
                left++;
            }
            
        }
        return count;
    }
}
