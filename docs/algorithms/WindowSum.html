604. Window Sum

Given an array of n integer, and a moving window(size k), move the window at each iteration 
from the start of the array, find the sum of the element inside the window at each moving.
Example

For array [1,2,7,8,5], moving window size k = 3.
1 + 2 + 7 = 10
2 + 7 + 8 = 17
7 + 8 + 5 = 20
return [10,17,20]

Idea to solve the problem

Sliding window

public class Solution {
    /**
     * @param nums: a list of integers.
     * @param k: length of window.
     * @return: the sum of the element inside the window at each moving.
     */
     // Total runtime 3227 ms. Your submission beats 91.00% Submissions!
    public int[] winSum(int[] nums, int k) {
        int len = nums.length;
        if ( len == 0 || k > len ) return new int[0];
        int knum = len - k + 1;
        int[] result =  new int[knum];
        result[0] = 0;
        for (int j=0 ; j < k; j++ ) 
                result[0] += nums[j];
        int index = 0, sum = 0;
        for (int i= 1; i < knum; i++) {
            result[i] = result[i-1] - nums[i-1] + nums[i+ k -1];
        }
        return result;
    }
}
