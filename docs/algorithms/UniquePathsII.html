115. Unique Paths II

Description of the Problem

Follow up for "Unique Paths":

Now consider if some obstacles are added to the grids. How many unique paths would there be?

An obstacle and empty space is marked as 1 and 0 respectively in the grid.
Example

For example,

There is one obstacle in the middle of a 3x3 grid as illustrated below.

[
  [0,0,0],
  [0,1,0],
  [0,0,0]
]

Idea to solve the problem

Dynamic Programming

public class Solution {
    /**
     * @param obstacleGrid: A list of lists of integers
     * @return: An integer
     */
    // Total runtime 8035 ms. Your submission beats 92.00% Submissions!
    public int uniquePathsWithObstacles(int[][] obstacleGrid) {
      if ( obstacleGrid == null || obstacleGrid.length == 0 ||
            obstacleGrid[0].length == 0 ) return 0;
      int rows = obstacleGrid.length;
      int cols = obstacleGrid[0].length;
      if ( obstacleGrid[rows-1][cols-1] == 1) return 0;
      int[][] dp = new int[rows][cols];
      dp[rows-1][cols-1] = 1;
      for (int j=cols-2; j >=0; j-- )
        if (obstacleGrid[rows-1][j] == 1)
          dp[rows-1][j] = 0;
        else 
          dp[rows-1][j] = dp[rows-1][j+1];
      for (int i=rows-2; i >=0; i-- )
        if (obstacleGrid[i][cols-1] == 1)
          dp[i][cols-1] = 0;
        else 
          dp[i][cols-1] = dp[i+1][cols-1];
      for (int i=rows-2; i>=0; i--) {
        for (int j=cols-2; j>=0; j--) {
          if ( obstacleGrid[i][j] == 1)
            dp[i][j] = 0;
          else
            dp[i][j] = dp[i+1][j] + dp[i][j+1];
        }
      }
      return dp[0][0];
    }
}
